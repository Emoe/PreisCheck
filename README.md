# PreisCheck
This tool can be used to check for low prices at:
* saturn
* mediamerkt
* cyberport
* notebooksbilliger

The tool checks for new price at 8 AM and PM.

# Installation
Install the requierments via:
``` pip3 install -r requierments.txt ```

# Use
Run the via Commandline with:
```python3 main.py ```
